THE APPROACH TAKEN

User Research Approach

To gain insights and inspiration for the project, I conducted user research across multiple platforms, including Dribble, Pinterest, Landmark, and other food delivery related websites. These platforms helped me understand current design patterns, user expectations, and visual trends in the industry.

In addition, I incorporated a question and answer approach, which allowed me to gather direct feedback and better understand user behaviors, pain points, and preferences. This combination of visual research and interactive inquiry played a key role in shaping the overall user experience.

Information Architecture

Create a clean and modern user flow diagram for a mobile or web-based e-commerce app. The flow should start from the Home page and guide users through Search, Filter, Cart, Payment, and Order Success, continuing to Order Tracking, History, and Re-Ordering. Each screen should include key CTAs like 'Discover your interest', 'Add to cart', 'Checkout', 'Select payment method', and 'Track order'. Use a simple black top bar for window representation and bright blue buttons for primary actions. Ensure directional arrows indicate navigation between pages.

Wireframe (still in progress)

A step-by-step low-fidelity wireframe for the product search flow in this mobile e-commerce app. Which includes 5 screens in a horizontal sequence showing the clickable areas and the user journey from search initiation to product review.

  • Screen 1: Home page with a search bar and product categories.
  • Screen 2: Search overlay with keyboard prompt activated and a highlighted input field.
  • Screen 3: Live search suggestions appear as the user types.
  • Screen 4: Display multiple search result cards; user selects one.
  • Screen 5: Product detail page where the user can view item info and add to cart.

Usability Testing & Design Critique

I collaborated closely with stakeholders, testers, and developers to identify key usability issues and improve the design through iteration. A/B testing with users helped validate design decisions and gather actionable feedback.

Regular design critique sessions allowed me to evaluate and refine visual hierarchy, consistency, and usability ensuring a user centered and effective final product.
